# qishiyundong
策马扬鞭、驰骋草原，这无疑是大多数人心中的期望。元其是见惯了城市的喧嚣，回归大自然更是无数人向往的目标。骑马有益人身心健康。
骑士运动主要功能有是马场第预定与马术信息新闻，骑马开拓视野、舒缓心情，更是能够通过自然消耗身体多余热量达到瘦身健身的效果。而且长期骑马还可以磨练人的意志。
